# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a GitHub Issue with the label `conduct`, or use [GitHub's private reporting](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam).

## Enforcement

Project maintainers will review and investigate all complaints and respond in a way that is appropriate to the circumstances. Further details of enforcement policies may be posted separately.
