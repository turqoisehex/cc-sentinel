# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest on `master` | Yes |
| Older releases | Best effort |

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/turqoisehex/cc-sentinel/security/advisories/new) to report security issues. This keeps the report confidential until a fix is available.

**Do not** open a public issue for security vulnerabilities.

## Response Time

This is a side project maintained in spare time. Critical vulnerabilities will be prioritized, but please allow reasonable time for a response.

## Disclosure

We follow coordinated disclosure — fixes will be released before public details are shared.
